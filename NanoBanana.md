# Stone Cold Scissors — Nano Banana Asset Pipeline

## Visual Style Guide
- **Art Style:** 16-bit retro arcade, pixel-art.
- **Outlines:** 2px solid black.
- **Lighting:** Top-left bias.
- **Animations:** Punchy, exaggerated, "squash and stretch" energy.

## Sprite Sheet Rules
- **Format:** Horizontal layout `[1][2][3][4][5][6]`.
- **Transparency:** Strictly transparent backgrounds.
- **Canvas:** Fixed size per frame, consistent anchor points (wrist position).

## Asset Implementation
- **Current Status:** Using SVG placeholders (Lucide-React) and CSS filters.
- **Next Step:** Swap with generated sprite sheets for:
    - **ROCK:** Clenched fist, forward punch.
    - **PAPER:** Flat palm, sweeping slap.
    - **SCISSORS:** Two-finger jab, snip motion.
    - **UI:** Arcade-style buttons and borders.
